# shell传参问题

```
#!/bin/bash
echo "命令：$0";
echo "参数1：$1";
echo "参数2：$2";
echo "参数3：$3";
echo "参数个数：$#";
echo "所有参数形成的字符串，和输入一直：$*";
echo "脚本运行的当前进程ID号：$$";
echo "后台运行的最后一个进程的ID号：$!";
echo "和\$*类似，只是每个参数用引号括起来了：$@";
echo "显示Shell使用的当前选项：$-";
echo "显示最后命令的退出状态码：$?";
```

    ./test.sh 1 2 3

    和$*类似，只是每个参数用引号括起来了：1 2 3
